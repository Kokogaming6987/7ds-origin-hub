Guides et Conseils

Maitrisez Seven Deadly Sins: Origin avec nos guides complets

Guide du Debutant

Commencer le Jeu

1. Le Prologue - Suivez l'histoire principale pour debloquer les bases du combat, les artefacts et les equipements.

2. Gachas Stratagiques - Gardez vos diamants pour les banners SSR garantiques. Ne depensez pas tout sur les banners temporaires.

3. Gestion des Ressources - Ne laissez pas vos stamina/energie au max. Utilisez-la regulierement pour farmer des materiaux.

4. Premieres Cibles - Concentrez-vous sur le leveling des personnages SSR en priorite. Les SR peuvent servir en early game mais seront remplaces.

Systeme de Combat

Elements et Faiblesses - Feau bat Eaux, Eaux bat Feu, Terre bat Foudre, Foudre bat Terre, Lumiere bat Obscurite, Obscurite bat Lumiere, Vent bat Froid, Froid bat Vent. Exploitez ces faiblesses pour infliger plus de degats.

Chain Combo - Enchainnez les attaques des personnages de meme element pour declencher des combo attacks bonus. 3+ chain donnent un bonus significatif.

Ultis - Les capacites Ulti se chargent pendant le combat. Utilisez-les au moment optimal pour maximiser les degats ou les buffs d'equipe.

Artefacts et Equipements

Sets d'Artefacts - Equipper 4 artefacts du meme set declenche un bonus de set. Les meilleurs sets end-game sont: Set Defense, Set Attack, Set Speed, Set Crit.

Stats Prioritaires - Tank: DEF > HP > SPD. DPS: ATK > Crit > Crit Dmg > SPD. Support: SPD > HP > DEF.

Upgrading - Ne montez les artefacts qu'a partir du niveau 30+. Avant, utilisez des artefacts de niveau inferieur pour economiser des ressources.

Farming et Ressources

Stamina Farming - Farmez toujours les donjons d'experience en priorite pour monter vos personnages. Les artefacts se farment dans les donjons specifiques.

Event Bonus - Participez a TOUS les evenements. Les bonus d'experience et de drop sont souvent doubles pendant les evenements.

Daily Missions - Completez toutes les quetes journalieres pour les diamants gratuits. C'est la source principale de gacha F2P.

Composition d'Equipe

Meta Team - Une equipe ideale a: 1 Tank, 2-3 DPS, 1 Support. Les DPS doivent etre du meme element pour maximiser les chain combos.

Synergie - Certains personnages buffent des stats specifiques. Assurez-vous que vos DPS beneficient de ces buffs.

Flexibility - Ayez au moins 2 equipes completes pour gerer le contenu multi-formation (Abyss, Arena, etc.).

Conseils End-Game

Abyss Mode - Adaptez votre equipe a chaque etage. Notez les elements ennemis et preparez votre contre.

Archi-Boss - Farmez les fragments de personnages dans les raids pour monter les evolutions.

Archi-PVP - La vitesse est REINE en PVP. Optimisez votre equipe pour avoir le plus de SPD possible.

Meta Tracking - Les meta evoluent avec chaque mise a jour. Suivez les buffs/nerfs des personnages.

FAQ - Questions Frequentes

Q: Quel SSR reroll au debut? - Ciblez Meliodas, Diane ou Jericho. Ce sont les plus polyvalents.

Q: Dois-je depenser de l'argent? - Non, le jeu est tres F2P friendly avec de nombreuses sources de gacha gratuit.

Q: Comment farmer efficacement? - Auto-farm les donjons experience le plus souvent possible. C'est le goulot d'etranglement principal.
