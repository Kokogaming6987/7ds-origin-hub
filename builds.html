Meliodas - Dragon Mark

DPS / Burst

Stats Prioritaires

  • 1. ATK: 3500+ (priorite absolue)
  • 2. CRIT: 80%+ pour garantir les coups critiques
  • 3. CRIT DMG: 250%+ (multiplicateur de degats)
  • 4. SPD: 200+ pour agir rapidement

Artefacts Recommandes

  • Set 1: Critique (4 pieces) - Augmente les coups critiques
  • Set 2: Attaque (2 pieces) - Renforce ATK brut
  • Reliques: Amplificateur Critique, Amplificateur Attaque

Rotation de Combat

  1. Tour 1: Charger le Dragon Mark (activer le transformateur)
  2. Tour 2: Attaque de base pour generer de la puissance
  3. Tour 3: Ultimate - Decime les ennemis avec le Dragon Mark active
  4. Tours suivants: Repeter le cycle ou attaquer si l'ennemi est pres de la mort

Elizabeth - Support

Support / Heal

Stats Prioritaires

  • 1. SPD: 250+ (agir en premier)
  • 2. DEF: 2000+ (tanker les degats)
  • 3. HP: 5000+ (survivre)
  • 4. CRIT: 50%+ pour quelques coups critiques

Artefacts Recommandes

  • Set 1: Vitesse (4 pieces) - Augmente SPD
  • Set 2: Vie (2 pieces) - Renforce HP
  • Reliques: Amplificateur Vitesse, Amplificateur Soin

Composition Ideale

Elizabeth fonctionne mieux en equipe avec des DPS frontaux et des tanks.

King - Controleur

CC / Control

Stats Prioritaires

  • 1. SPD: 240+ (controler l'ordre)
  • 2. ACC: 70%+ (ne pas manquer les pieges)
  • 3. DEF: 1800+ (survie)
  • 4. HP: 4500+

Artefacts Recommandes

  • Set 1: Vitesse (4 pieces)
  • Set 2: Defense (2 pieces)
  • Reliques: Amplificateur Vitesse, Amplificateur Precision

Objectifs du Build

  • Immobiliser les ennemis avant qu'ils n'agissent
  • Creer de l'espace pour que les DPS frappent
  • Survivre assez longtemps pour tenir vos effets

Ban - Tank Regeneratif

Tank / Sustain

Stats Prioritaires

  • 1. HP: 6000+ (survie maximale)
  • 2. DEF: 2200+ (reduction de degats)
  • 3. SPD: 180+ (temps de regeneration)
  • 4. RES: 60%+ (resister aux effets)

Artefacts Recommandes

  • Set 1: Defense (4 pieces)
  • Set 2: Vie (2 pieces)
  • Reliques: Amplificateur Defense, Amplificateur Vie

Synergie Ideale

Ban excelle avec Elizabeth pour une combinaison tank + heal infaillible.

Guide des Stats

ATK (Attaque)

Augmente les degats de base. Essentiel pour tous les DPS.

CRIT (Critique)

Chance de coup critique. 80% est l'ideal pour les DPS.

CRIT DMG (Degats Critique)

Multiplicateur de degats sur les coups critiques. 250%+ recommande.

SPD (Vitesse)

Determine l'ordre d'action. Plus la valeur est haute, plus vite vous agissez.

DEF (Defense)

Reduit les degats subis. Essentiel pour les tanks.

HP (Points de Vie)

Sante globale. Toujours utile pour la survie.

ACC (Precision)

Chance de toucher. Important pour les controles.

RES (Resistance)

Resistance aux effets negatifs. Protege contre les debuffs.
